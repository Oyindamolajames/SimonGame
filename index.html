At this point, it might be worth reviewing how the Simon game works.

Firstly, the game shows the first colour in the sequence (blue). You then click on the blue button.

Next, the game shows the next colour (red), you have to remember the sequence is blue, red and so on and so forth.

If you mess up the sequence, then the game ends.
